This a course management system for Giki written in C++ using data structures.It has developed by a team of six members. File system is used for storing data. It 
has two views.

1) Student
  i) He/She can register, add and drop a course.
  ii) He/She can show his/her course status.
  iii) He/She can generate the course list.
   iv) He/She can change password and can remove unregistered courses.

2) Admin
  i) It can add a new admin member and remove an admin member.
  ii) It can generate an admin list.
  iii) It can add and remove a course.
   iv) It can add and remove a student.
    v) It can generate student and course list.
    vi) It can approve the registration, add and drop request from the student.
